# Chatbot de Comandos Docker 🐳

Asistente virtual sencillo desarrollado en Python para la consulta rápida de comandos de Docker. Utiliza una interfaz gráfica amigable y una base de datos basada en JSON.

## 🚀 Funcionalidades
- **Interfaz Gráfica**: Ventana de escritorio con historial de conversación y scroll.
- **Base de Datos JSON**: Fácil de actualizar y mantener sin modificar el código fuente.
- **Motor de búsqueda**: Lógica basada en coincidencia de palabras clave para encontrar la mejor respuesta.

## 🛠️ Requisitos
- **Python 3.x**
- **Tkinter**
